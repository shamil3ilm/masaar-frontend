Button

Button.tsx

Sizes
Icon slots & loading
cva config 
import { cva } from "class-variance-authority";

export const button = cva(
  "inline-flex items-center justify-center gap-2 font-medium rounded-[--radius] " +
  "transition-colors disabled:opacity-50 disabled:pointer-events-none " +
  "focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-brand focus-visible:ring-offset-2 " +
  "focus-visible:ring-offset-bg",
  {
    variants: {
      variant: {
        primary:   "bg-brand text-brand-fg hover:bg-brand-dark",
        secondary: "bg-surface-2 text-text hover:bg-border",
        outline:   "border border-border-strong text-text hover:bg-surface-2",
        ghost:     "text-text hover:bg-surface-2",
        danger:    "bg-danger text-danger-fg hover:opacity-90",
      },
      size: {
        sm: "h-8  px-2.5 text-xs",
        md: "h-9  px-3.5 text-sm",
        lg: "h-11 px-5   text-[15px]",
      },
    },
    defaultVariants: { variant: "primary", size: "md" },
  }
);

Step 05

The application shell

A dark-navy sidebar (in both themes), a topbar with profile + theme + direction controls, and a page header. Sidebar collapses to a rail on lg+, becomes an off-canvas drawer below lg.

Sidebar

Dark navy, in any theme

Grouped nav with lucide icons; active item gets a tinted background + 2px inset start border; collapses to a 64px rail.

TopBar

Workspace, theme, dir, profile

Breadcrumb slot, theme + direction toggles, prominent profile menu (avatar · name · role).

PageHeader

Title + actions + back

Optional back button uses logical icon flip; primary action is end-aligned.
